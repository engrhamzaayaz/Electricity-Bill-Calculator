# ⚡ Electricity Bill Calculator

A smart electricity billing system developed as part of academic 
coursework at **PIEAS, Islamabad**.

## 📌 About
This project designs an efficient system to calculate electricity 
bills based on unit consumption, applying slab-wise tariffs, 
surcharges, and other real-world billing components.

## 🎯 Objectives
- Understand real-world electricity billing mechanisms
- Strengthen programming and problem-solving skills
- Bridge theoretical concepts with real-life applications

## ✨ Features
- Slab-wise tariff calculation
- Surcharges and additional billing components
- Conditional logic for accurate bill generation
- Simple console-based user interface

## 🛠️ Built With
- C++

## 🚀 How to Run
1. Clone or download the project
2. Open in any C++ compiler (e.g. Code::Blocks, VS Code)
3. Compile the `.cpp` file
4. Run and enter units consumed when prompted
